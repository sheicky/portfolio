A few things I've built. The rest are on my GitHub. Most have a short demo just below.

SaChi-ASR — I fine-tuned OpenAI's Whisper on Mooré, my first language and a low-resource African language spoken in Burkina Faso, then put the model on Hugging Face so native speakers can use it. Model
Autonomous ML Agent — an LLM-driven AutoML platform that tunes hyperparameters with Optuna and trains several models in parallel, then keeps the best one. Code
TradeAI — a trading assistant that predicts market moves from 40+ global indices, including gold, the VIX, and bond rates, with a Random Forest model. Code
Stock Analysis with LLM — find stocks from plain-English queries like "companies that build data centers," backed by embeddings over company data. Code
LLM Evaluation Platform — a Streamlit and DeepEval app that benchmarks Gemini against LLaMA in real time and scores coherence, hallucination rate, and RAG quality. Code
AI Answer Engine — a Perplexity-style chat that answers from YouTube, PDFs, CSVs, images, and web pages with inline citations, on Gemini Flash 2.0. Code
Codebase RAG — ask questions about a whole codebase and get answers grounded in the actual files. Code
AI Cursor Clone — a small AI code editor in the style of Cursor, with inline chat and edits. Code
AI Chrome Extension — a browser extension that drops an AI assistant onto the page you're reading. Code
Customer Churn Prediction — a Streamlit app that predicts bank customer churn with Random Forest, XGBoost, and KNN. Code
Pentagram — self-hosted SDXL-Turbo image generation on Modal GPUs, no vendor API, about two seconds end to end. Code
