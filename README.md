# portfolio

Personal portfolio site for Sheick Ali Simpore — a minimal, static, single-page site
inspired by [karpathy.ai](https://karpathy.ai). Hosted on GitHub Pages, deployed via
GitHub Actions.

## Stack

- Static `index.html` + `style.css` (no framework, no build step)
- Vanilla JS for the light/dark toggle
- GitHub Pages + GitHub Actions for CI/CD

## Development

Open `index.html` directly in a browser, or serve locally:

```bash
python3 -m http.server 8000
```
